Expense Tracker Application using React, React-Router, Redux, ECMAScript 6, CSS3, Jest and Enzyme

• Implemented to add functionalities through which user can add expenses, edit expenses and remove expenses.
• Used React Router to allow the user to navigate between different components.
• Added text and date filters, and sorting options to allow the user to get a view of desired expenses in sorted order.
• Added test cases for different components using Jest testing tool and Snapshot testing using Enzyme tool.
• Tested user interaction by simulating events.
